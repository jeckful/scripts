# Scripts Collection

Useful bash scripts for various tasks.

## Available Scripts

### dotfiles_sync
Generic synchronization script for managing dotfiles across machines.

### discord_updater
Update Discord on Debian/Ubuntu systems using the official `.deb` package.

## Usage

Each script is in its own directory with:
- The executable script file
- A dedicated README with specific instructions
